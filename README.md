# NAS随身听

NAS随身听是一款第三方私有 NAS 音乐播放器，支持通过远程访问地址、FN Connect、WebDAV 等方式连接自己的 NAS，浏览并播放个人音乐库。

支持飞牛 fnOS / FN Connect / WebDAV 等私有 NAS 环境。本项目为第三方开源工具，非官方应用。

本应用为第三方 NAS 音乐播放器，非飞牛官方应用，未与飞牛官方建立授权、合作或从属关系。应用仅用于连接用户自己的 NAS 并播放个人音乐文件。

This app is an independent third-party NAS music player. It is not an official Feiniu/fnOS app and is not affiliated with, endorsed by, or authorized by Feiniu. It is designed to connect to the user's own NAS and play personal music files.
